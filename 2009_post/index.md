# 做人难，做好人更难

今天又是心情复杂的一天。最近总是感到混乱和痛苦，好像被什么东西压着，喘不过气来。

说起来有些矛盾，之前觉得挺喜欢A的，但最近帮助她处理工作的事情让我感到很疲惫。她有ASD，我知道她有她的难处，我的共情能力让我很容易察觉到她的需要，也习惯性地去帮助她。但是这种帮助正在消耗我，让我甚至有些开始不喜欢她了。

更让我难受的是，我付出了心力和脑力，却好像是理所当然的事情。有时候我会想，我也有我的难处啊，但好像没有人会像我琢磨别人那样来理解我。

我一直认为善良是重要的，应该帮助别人，理解别人。但现在我现在的状态其实并不适合帮助别人——情绪混乱、时间不够、对A产生负面情绪。在这种状态下强行帮助，可能对谁都不是真正的好。为了能够长久地保持善良。我可以关心别人，但不必为别人的每个情绪负责。

接下来想做的：
学会诚实地表达自己的困难
给自己设定帮助他人的时间边界
提醒自己：帮助是选择，不是义务
对自己温柔一点，我已经做得够好了

今天的思考让我好受了一些。改变不会一蹴而就，但至少我知道了方向。
